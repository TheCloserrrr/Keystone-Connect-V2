
# Keystone Connect — Next.js (App Router) + Tailwind

## Local Dev
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
npm start
```

## Deploy (Vercel dashboard)
1. Push this folder to a new public GitHub repo.
2. Go to https://vercel.com/import (or click "Add New Project").
3. Import your repo. Framework should auto-detect: **Next.js**.
4. Click Deploy. Your site will be live on a vercel.app URL.
5. Add a custom domain (e.g., keystoneconnect.co) in **Domains**.

## Deploy (Vercel CLI)
```bash
npm i -g vercel
vercel
vercel --prod
```
